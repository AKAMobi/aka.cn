阿卡直通车服务常见问题
  1. 怎样注册用户？
    2. 请登陆http://my.aka.com.cn/，选择注册新用户即可。

  2. 用户发展下线有什么奖励？
    3. 下线在进行网上支付后会给你 5% 的额外奖励，下线的下线网上支付后会给你10%的额外奖励， 所以，赶快拉你的朋友来注册吧！记得注册的时候在“您是如何得知VPN服务的？” 中改为“别人介绍的”，然后里面添写你的ID!
  3. 如何使用阿卡直通车（VPN版）？
    4. 请看：一步一步配置直通车VPN版
  4. 如何使用阿卡直通车(Proxy版)？
    5. 请看：一步一步配置直通车Proxy版
  5. 我想使用阿卡直通车（Proxy版），为什么总无法通过IE的认证？
    6. 首先确保你的账号/密码正确，然后确认在 http://my.aka.com.cn/ 里面开启了 “阿卡直通车(Proxy)版”的功能。
    如果还无法使用阿卡直通车，请联系阿卡公司。
  6. 在网上哪里有讨论阿卡直通车的讨论区？
    7. 大话西游（bbs.zixia.net）有专门的阿卡直通车讨论区， 请登陆大话西游BBS，然后选择“分类讨论区”中的“1 大话西游 [大话/技术]”，然后选择PersonalVPN版参加讨论。
  7. 阿卡直通车的价格是多少？

    8. 阿卡回国直通车（VPN包月版）标准定价88元每月；
    阿卡出国直通车（VPN包月版）标准定价28.8元每月；
    阿卡直通车（VPN计时版）标准定价0.04元每分钟，每月封顶176元；
    阿卡直通车(Proxy版）标准定价55元每月。

    2002年11月至2003年8月期间阿卡直通车进行优惠，打六折， 价格分别为：

    阿卡回国直通车（VPN包月版）标准定价88 * 0.6 = 52.8元每月；
    阿卡出国直通车不参与本次打折活动；
    阿卡直通车（VPN计时版）标准定价0.04 * 0.6 = 0.024元每分钟，每月封顶176 * 0.6 = 105.6元；
    阿卡直通车(Proxy版）标准定价55 * 0.6 = 33元每月。

  8. 为什么在（http://www.aka.com.cn/solution/vpn/edu/price.shtml）中又有列表， 最少的512Kb带宽的包月费是700元？如果是教育网内的个人用户，收费到底是多少？
    9. 阿卡直通车（VPN版）包月费为88元，在price.shtml中的价格是企业版的价格， 和个人用户无关。 企业出国VPN是专门为高校网络中心提供的对普通用户完全透明的VPN出国 解决方案，也可以为一个实验室的局域网提供服务。
  9. 若想采用计时封顶的办法，速度多快，质量能不能保证？多长时间结算一次？ 费用是0.04元/分钟吗？每月有没有最低消费的限制？封顶计费周期怎样算？
    10. 阿卡直通车的速度不慢于教育网直接出国，商业服务当然首先要保证质量。 计时版结算时间为你每次使用后，按照使用时间长度收费。 计时版每月费用封顶计算日期为每个月的一号到下个月的一号。
  10. 付款方式是怎样的？是只能采用网上支付吗？
    11. 除了网上支付，目前本公司也在发展代理，可以找你最近的代理交费。
    如果你没有网上支付卡，也可以找有网上支付卡的朋友帮忙。
  11. 阿卡直通车怎样在国外交费啊？
    12. 阿卡的网上支付支持几乎国内、国外所有的银行卡、信用卡，包括VISA/MASTER国际信用卡。
  12. 可以随时终止服务吗？短期用户，如一两个月的，可以吗？
    13. 完全没有问题。
  13. 有开户费吗？账户上最低要保持多少钱？
    14. 开户费全免，帐户上没有最低留存金额的限制。
  14. 阿卡直通车（VPN版）对网络防火墙有什么要求吗？
    15. 阿卡直通车需要防火墙开放1723/tcp端口的访问， 并允许类型为GRE（47）的IP包通过。
  15. 我不安装阿卡直通车客户端可以使用阿卡直通车吗？
    16. 不可以。
    阿卡直通车目前只能使用 阿卡直通车专用客户端进行连接使用， (说明页面在这里 ) 如果不使用客户端，本公司不负责解释网络不通、断网、感冒、 发烧、流鼻涕打喷嚏、女友分手、考试不及格、火山爆发、地震 等任何异常问题。
  16. 我可以用阿卡直通车开设Proxy或将其作为网关，为多人提供代理吗？
    17. 绝对不可以。
    这种情况建议你购买阿卡企业出国 VPNhttp://vpn.aka.com.cn/。 因为阿卡直通车是禁止级连的，这样对别的用户是一种不公平， 如果发现用户有这种行为，用户帐号会被直接封禁而不退还任何费用。
  17. 我想办理一张支持网上支付的银行卡……
    18. 可以寻找最近的招商银行，办理一卡通。
    清华东门新开了一家招商银行： 出东门向南走 100 米路西，清华紫光新楼一层。 先拿号，然后窗口办理开户，会发给你一个用户手册， 同时可在网上办理开通网上支付。
  18. 我在教育网用直通车，朋友在美国，想用MSN语音聊天。 但总是告诉我“无法与对方计算机进行连接”，不过我用直通车和在国内 公众网的朋友语音对话就没有问题，请问有什么办法解决这个问题？
    19. 阿卡直通车支持MSN语音与视频聊天，只需要国内的用户使用出国直通车，并且国外的用户使用回国直通车即可。
