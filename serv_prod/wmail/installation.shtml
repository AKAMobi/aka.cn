安装说明
qmail/vpopmail/ucspi-tcp/maildrop的安装说明请自己参考软件中的说明文档。安装vpopmail时，推荐将vpopmail的uid改成与apache的uid相同，并用apache组用户。maildrop请安装到默认的/usr/local/bin目录下。
启动smtp，用vchkpw做smth-auth的命令行：

/usr/local/bin/tcpserver 0 smtp /var/qmail/bin/qmail-smtpd \
lfan.net /home/vpopmail/bin/vchkpw /bin/true 2>&1 \
| /var/qmail/bin/splogger smtpd 3 &

安装wmail时，推荐用./install安装，安装目录是/home/wmail 你也可以根据需要修改install文件，让它安装到指定目录。

安装完毕后修改httpd.conf，加上以下内容，注意替换必要的内容：

<VirtualHost *:80>
SetEnv SQWEBMAIL_TEMPLATEDIR /home/wmail/html
ScriptAlias /cgi-bin/ "/home/wmail/www/cgi-bin/"
<Directory "/home/wmail/www">
AllowOverride All
Options +Includes
</Directory>
ServerAdmin lfan@lfan.net
DocumentRoot /home/wmail/www
ServerName mail.xxx.com
</VirtualHost>
